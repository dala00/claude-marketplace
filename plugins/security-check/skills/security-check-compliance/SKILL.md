このプロジェクトの法務・コンプライアンス面をチェックしてください。以下の項目を確認すること：

- **電気通信事業の届出**: プライベートチャット、メッセージング、通話などユーザー間の通信機能が含まれる場合、電気通信事業の届出が必要かどうかを判定する
- **特定商取引法に基づく表記**: 有料サービス・商品の販売がある場合、特定商取引法に基づく表記が必要かを判定する。必要な場合は表記内容の妥当性もチェックする
- **利用規約・プライバシーポリシー**: ユーザー登録、個人情報の取得、Cookie の使用などがある場合、利用規約やプライバシーポリシーのページが必要かを判定する
  - ただし文書の内容の網羅的なチェックは量が多いため自発的には行わない。ユーザーから要望があればチェックするが、その際は法的な完全性を保証するものではないことを明示すること
- **クレジット・ライセンス表記**: 使用しているOSSライブラリ、フォント、アイコンなどのライセンスに基づくクレジット表記が適切に行われているか確認する
- **商標・知的財産権**: 他者が権利を持つ商標、ロゴ、ブランド名を無断で使用していないか確認する。画像やアイコンなどのアセットも含む

検出結果は重大度レベル（Critical、High、Medium、Low、Info）で報告し、各問題に対して具体的な対応方法を提示すること。
